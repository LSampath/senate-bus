This is a synchronized solution for the Senate Bus Problem.
The application consists of five classes.

######################################################################

Assignment.java >>>>> Main class for the Senate Bus problem
Bus.java >>>>> Class for the Bus threads
BusScheduler.java >>>>> Class for creating threads for the bus arrival
Rider.java >>>>> Class for the Rider threads
RiderScheduler.java >>>>> Class for creating threads for the riders arrival
SharedResources.java >>>>> Initialization

#####################################################################

To run the Application simply use the run button.

#####################################################################
				                or
#####################################################################
Senate-Bus.jar is attached in the project directory.
Go to the Project directory.
Run Senate-Bus.jar file using the following command
	
	java -jar Senate-Bus.jar
